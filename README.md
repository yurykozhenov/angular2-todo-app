# [Angular 2 Todo App](https://yk-angular2-todo-app.herokuapp.com)
Basic Angular 2 application

### Requirements:
- NodeJS >= 6.9.2

### Commands:

Install dependencies:
```
npm install
```

Run app in debug mode:
```
npm run debug
```

Build app:
```
npm run build
```

Run builded app:
```
npm start
```
